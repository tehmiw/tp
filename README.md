* **CoursePilot** is a desktop application for university tutors and teaching assistants to manage tutorial groups, track student participation, and handle grading progress.
* It is optimized for fast typists who prefer a CLI-based workflow, while still providing a GUI for visual feedback.
* For the detailed documentation of this project, see the **[CoursePilot Product Website](https://ay2526s2-cs2103t-w13-4.github.io/tp)**.
* This project is based on the AddressBook-Level3 project created by the [SE-EDU initiative](https://se-education.org).
